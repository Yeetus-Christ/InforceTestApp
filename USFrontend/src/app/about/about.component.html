About the URL Shortener

Welcome to the URL Shortener! This service allows you to create shorter, more convenient URLs from longer ones.

How the Shortening Algorithm Works

The algorithm works by taking the original URL and generating a unique short version by encoding the original URL's hash value using a base62 encoding scheme.
The base62 encoding uses a set of characters (0-9, a-z, A-Z) to create a shortened string. This string is then prepended with a domain (e.g., https://short.ly/) to create the final short URL.
